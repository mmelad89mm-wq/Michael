# 📚 برنامج إدارة الدروس التعليمية (كامل الخصائص)

نظام متكامل لإدارة المعلمين، الطلاب، الحضور، الاشتراكات، الدرجات، والتواصل مع أولياء الأمور عبر واتساب. يعمل بدون إنترنت (Offline First) ويحفظ البيانات في المتصفح.

## ✨ المميزات الرئيسية
- إدارة المدرسين والصفوف والمجموعات (مواعيد)
- تسجيل الطلاب مع رقم ولي الأمر
- نظام حضور/غياب مع حساب النسبة المئوية
- إدارة الاشتراكات الشهرية (مدفوع / غير مدفوع) وعرض المتأخرين
- درجات الاختبارات (شهري، مفاجئ، نهائي) وحساب المتوسط
- تقارير شاملة: حضور، غياب مستمر، اشتراكات، درجات
- التواصل المباشر عبر واتساب (رسائل مخصصة، تنبيهات غياب، إشعارات)
- تصدير إلى Excel
- واجهة عربية بسيطة وجذابة

## 🛠️ كيفية التشغيل
1. حمل الملفات `index.html` و `app.js` في مجلد واحد.
2. افتح `index.html` بأي متصفح حديث (Chrome, Edge, Firefox).
3. جميع البيانات تحفظ تلقائياً في `localStorage` لن تفقدها بعد إغلاق المتصفح.

## 📂 هيكل الملفات
- `index.html` : الواجهة الرئيسية (HTML/CSS)
- `app.js` : كود الجافاسكريبت بالكامل (منطق البرنامج)
- `README.md` : هذا الملف

## 🔧 ملاحظات
- لاستخدام خاصية الواتساب، يجب أن يكون رقم ولي الأمر مكتوباً بالصيغة الدولية (مثال: 9665xxxxxxxx).
- خاصية التصدير إلى Excel تعمل فوراً بضغطة زر.
- يمكن إضافة أكثر من مدرس وربطه بالمجموعات (يظهر في القوائم).

## 🚀 مستقبلاً
يمكن تطويره ليدعم السحابية (Backend) لكن النسخة الحالية تغطي كل المتطلبات بشكل كامل.
